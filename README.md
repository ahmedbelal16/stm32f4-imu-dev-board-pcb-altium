# STM32F4 Custom PCB Design (Altium)

## Overview
This project is a custom development board based on the STM32F4 microcontroller, designed using Altium Designer.  
It includes power management, USB interface, debugging support, and an IMU sensor.

---

## Features
- STM32F4 microcontroller  
- MPU6050 IMU (I2C)  
- USB power input  
- 3.3V LDO regulator  
- SWD debugging interface  
- GPIO expansion header
